Welcome to Hvuntokrul's Excellent URL Shortener Service!

Use "https://www.freecodecamp.com" link format with the API:
https://fcc-link.herokuapp.com/api/https://www.freecodecamp.com

And you will get a response similar to:
{"original": "http://www.freecodecamp.com", "short": "https://fcc-link.herokuapp.com/5098"}

Now you can use the short link ID, so by going to:
https://fcc-link.herokuapp.com/5098

You will be redirected to:
http://www.freecodecamp.com
